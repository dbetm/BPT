Otros temas

VECTORES

Es una cantidad que describe un parámetro físico posee una magnitud, dirección y sentido, se puede representar de manera gráfica o analítica.

Analíticamente describe la proyección con cada eje que constituye el sistema de coordenadas donde se encuentra el vector.

ESCALAR

Representación de una propiedad física invariante ante transformación de coordenadas como una magnitud.

SISTEMA DE COORDENADAS

Un punto de origen y un conjunto de ejes que parten del origen que cumplen una relación. Un punto sobre este sistema se referencia como (X, Y, Z).

CENTRO DE MASA

El centro de masa de un cuerpo o un conjunto de cuerpos es el punto que se mueve como si toda la masa estuviera centrada ahí y todas las fuerzas se aplican sobre este punto.

MOMENTO LINEAL

El momento lineal es la cantidad de movimiento de una partícula u objeto. La rapidez de cambio de momento lineal es igual a la fuerza neta que actúa sobre ella y va en la dirección de una fuerza.
